Language Translation Options

Code Translation Description
th - en Thai to English Translate from Thai to English
en - th English to Thai Translate from English to Thai
th - zh Thai to Chinese Translate from Thai to Simplified Chinese
zh - th Chinese to Thai Translate from Chinese to Thai
th - ja Thai to Japanese Translate from Thai to Japanese
th - ar Thai to Arabic Translate from Thai to Arabic
th - ru Thai to Russian Translate from Thai to Russian
th - de Thai to German Translate from Thai to German
th - fr Thai to French Translate from Thai to French
th - es Thai to Spanish Translate from Thai to Spanish
th - vi Thai to Vietnamese Translate from Thai to Vietnamese

With cURL
curl -X 'GET' \
            'https://khain.app/api/translate?text=Hello&lang=en&to=th' \
            -H 'Accept: application/json' \
            -H 'Authorization: Bearer your-secret-token'
With Python
import requests
          
          url = "https://khain.app/api/translate"
          params = {
              "text": "Hello",
              "lang": "en",
              "to": "th"
          }
          headers = {
              "Accept": "application/json",
              "Authorization": "Bearer your-secret-token"
          }
          
          response = requests.get(url, params=params, headers=headers)
          print(response.json())
With Node.js
const fetch = require('node-fetch');
          
          const url = new URL('https://khain.app/api/translate');
          url.searchParams.append('text', 'Hello');
          url.searchParams.append('lang', 'en');
          url.searchParams.append('to', 'th');
          
          fetch(url, {
            method: 'GET',
            headers: {
              'Accept': 'application/json',
              'Authorization': 'Bearer your-secret-token'
            }
          })
            .then(response => response.json())
            .then(data => console.log(data))
            .catch(error => console.error('Error:', error));

Translate text between languages
Parameters:
- text: The text to translate (required)
- lang: Source language code (example: 'th')
- to: Target language code (example: 'en')
- Authorization: Bearer token (required in header)
